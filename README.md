# ⚖️ Global Legal Assistant (Microsoft Word Add-in)

إضافة ذكية، حرة، وديناميكية بالكامل لبرنامج مايكروسوفت وورد (Office Word Add-in) مصممة كبوابة تفاعلية شاملة لمساعدة رجال القانون والمستخدمين في البحث الآلي في القواميس واستدعاء نصوص الفصول والمواد القانونية لأي دولة في العالم وحقنها مباشرة داخل المستندات.

## 🎯 مميزات عمل الأداة (Core Technical Features)
تم تصميم هذه الإضافة بمعمارية مرنة ومفتوحة بالكامل (Fully Dynamic Engine) لتتجاوز قيود القواميس الصلبة؛ حيث تتيح الواجهة التفاعلية:
1. **البحث المفتوح (Universal Querying)**: يكتب المستخدم أي كلمة قانونية يدوياً أو رقم مادة (مثال: عقد، إيجار، طلاق، شراكة) دون أي شروط مسبقة للكود.
2. **الذكاء الجغرافي التلقائي (Locale Auto-Detection)**: التقاط لغة وسياق مستند الـ Word المفتوح تلقائياً عبر دالة `Office.context.displayLanguage` لتوجيه بوصلة البحث نحو الدولة المستهدفة (تونس، مصر، السعودية، فرنسا) آلياً.
3. **تطويع المسميات التشريعية (Terminology Adaptation)**: يقوم المحرك بتعديل المخرجات مع معايير الدولة آلياً (يسميها 'فصل' لتونس، و'مادة' لمصر والسعودية، و'بند' للقوانين الدولية).
4. **محرك حقن البيانات (Data Injection Engine)**: إدراج وحقن نص المادة القانونية المستخرجة تلقائياً عند موضع مؤشر الفأرة الحالي للمستخدم داخل صفحة الـ Word بضغطة زر واحدة عبر دالة `Word.run()`.

## 📂 بنية المستودع (Repository Structure)
* `manifest.xml`: الملف الإداري والتكويني لتحديد هوية الإضافة داخل متجر مايكروسوفت أوفيس وضبط الصلاحيات واللوحة الجانبية (`TaskPaneApp`).
* `taskpane.html`: واجهة المستخدم والمحرك الديناميكي المستقل (جافاسكريبت وHTML) المسؤول عن معالجة المدخلات، الاتصال بالبوابات التشريعية، وتسيير آليات الحقن.
